The scope for an mvp of kitsoki + built-in stories:

- Dev Story
    - Tickets (in repo, github)
    - PRs
    - Workspaces (Active Bugs, Features)
    - Git (bitbucket/git tool)
        - Rebase and resolve conflicts (fetch upstream)
        - Search for and clone repo
    - Self-Improvement
    
- Ticket search story
    - Assign parent query (assignee, component, etc...)
    - New bugs
    - Priority listing
    - Free text searching

- Bug fix story
    - Reproduce and Evaluate
        - UI (playwright)
        - API (python)
        - Security evaluation
    - Plan
        - Implementation
        - Docs/Specs
        - Tests
    - Implement
    - Validate & Recycle
    - Review & Recycle
